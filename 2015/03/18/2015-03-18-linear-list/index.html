线性表（linear list）

  • Data Structures
Programming

线性表是一个线性结构，由n（n≥0）个数据元素（结点）a[0]，a[1]，a[2]，…，a[n-1]组成的有限序列。

线性表的逻辑结构简单，便于实现和操作。因此，线性表在实际应用中被广泛采用的一种数据结构。

线性表的作用

在实际应用中，线性表都是以栈、队列、字符串、数组等特殊线性表的形式来使用。

线性表的特征

  1. 集合中必存在唯一的一个“第一元素”
  2. 集合中必存在唯一的一个“最后元素”
  3. 除最后一个元素之外，均有唯一的后继(后件)
  4. 除第一个元素之外，均有唯一的前驱(前件)

线性表的存储结构

  1. 顺序存储结构：顺序表（物理上连续）
  2. 链式存储结构：链表（物理上不连续，逻辑上连续）
