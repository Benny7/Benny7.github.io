堆排序（heap sort）

  • Algorithms
Programming

堆排序是利用堆积树（堆）这种数据结构所设计的排序算法。本质上堆排序是选择排序的一种。

堆

堆（二叉堆）是一个数组，可以被看成近似的完全二叉树，树节点对应数组元素。

图示：

堆的形式

  • 最大堆（满足最大堆性质）
    除根外所有节点i满足：A[PARENT(i)] >= A[i]
  • 最小堆（满足最小堆性质）
    除根外所有节点i满足：A[PARENT(i)] <= A[i]

堆排序的基本过程（最大堆）

  • MAX-HEAPIFY(A, i)：维护最大堆性质（$O(lgn)$）
  • BUILD-MAX-HEAP(A)：从无序数组中构建最大堆（$O(n)$）
  • HEAP-SORT(A)：进行原址排序（$O(nlgn)$）
