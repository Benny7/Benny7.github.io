OpenGL编程管线

  • Computer Graphics
Programming

基本概念

  • 图元（entity）：组成图像的基本单元（点，直线，多边形，图形，位图）
  • 片元（primitive）：像素点（像素，位置，法向量等）

OpenGL编程管线

  • 指定几何对象

  • 顶点处理（Vectex Shader）
    顶点（位置，纹理坐标，颜色）

  • 图元装配（Geometry Shader）
    组装图元

  • 图元处理
    裁剪，消隐

  • 栅格化
    图元数据栅格化为片元

  • 片元处理（Fragment Shader）
    处理像素颜色

  • 逐个片元操作
    剪切（glScissor），Alpha测试（glAlphaFunc），模版测试（glStencilFunc），深度测试（glDephtFunc），混合（glBlendFunc）

  • 帧缓冲操作
    帧缓冲写入
