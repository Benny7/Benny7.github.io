计数排序（counting sort）

  • Algorithms
Programming

计数排序是一种非基于比较的排序算法。

基本思想

对输入序列中的某个元素x，统计不大于x的元素的个数n，则x就可直接放在第n+1个位置（$x \in [0, k]$）。

图示

伪代码

时间复杂度：$T(n) = O(n + k)$
当$k = O(n)$时，$T(n) = O(n)$

选择排序是稳定的排序算法。
