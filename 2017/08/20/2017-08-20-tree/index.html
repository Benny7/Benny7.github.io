树（tree）

  • Data Structures
Programming

树是一种抽象数据类型，用来模拟树状结构性质的数据集合。它是由n（n>0）个有限节点组成的具有层次关系的集合，任何具有层次关系的问题都可以用树结构来表示。

特点

  • 每个节点有n（n>=0）个子节点
  • 没有父节点的节点为为根节点
  • 每一个非根节点都只有一个父节点
  • 除了根节点外，每个子节点都可分为多个互不相交的子树

术语

  • 度：节点的子树个数称为节点的度，最大节点的度称为树的度
  • 叶节点：度为0的节点
  • 分支节点：度不为0的节点
  • 高度：树中节点的最大层次称为树的高度
  • 森林：m（m>=0）棵互不相交的树的集合
  • 节点关系：祖先，子孙，父节点，子节点，兄弟节点，堂兄弟节点

树的种类

  • 无序树：子节点之间无顺序关系
  • 有序树：子节点之间有顺序关系（二叉树就是有序树）
