二分查找（binary search）

  • Algorithms
Programming

二分查找是一种在有序表（逻辑结构上有序）查找特定元素的查找算法

基本思想

查找过程中每次将查找范围缩小一半，直到最终确定是否存在目标元素

时间复杂度：$T(n) = O(lgn)$

理解

二分查找的关键在于快速定位中间值并缩小问题规模，所以像顺序表这种存储结构上有序数据结构很适合使用二分查找，链表由于无法快速定位中间值，所以不适合使用二分查找，顺序表虽然查找快速，但插入很慢，很多时候我们即需要高效的查找效率，同时也需要高效的插入和删除效率，这样一种即支持二分查找又拥有链式结构插入删除效率的数据结构就是二分查找树
