二叉搜索树
二叉搜索树（BST）是一种将二分搜索算法思想作用在链式结构上的数据结构，即保留链式结构节点操作的灵活性，同时兼具二分搜索的搜索效率

特点

  • 每个有左子树的节点的值都大于其左子树所有节点的值
  • 每个有右子树的节点的值都小于其右子树所有节点的值
  • 非叶子节点的左子树和右子树都是二叉搜索树
  • 没有节点的值相同

删除

  1. 删除节点没有子节点，则直接删除
  2. 删除节点有一个子节点，则用子节点替代删除节点
  3. 删除节点有两个子节点，则找到删除节点的后继节点替代删除节点

优点

二叉搜索树的优点在于搜索，插入效率较高（$T(n) = (lgn)$）

理解

二叉搜索树的算法的运行时间取决于树的形状，二叉搜索树越平衡（越趋向完美二叉树）算法效率越高
